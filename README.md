## Features

- **Zero fees** — no gas costs for users
- **Instant** — ~0.7s transaction finality
- **Self-custody** — payments go directly to your wallet
- **QR payments** — scan and pay from any wallet
- **Real-time** — WebSocket status updates

## Stack

React + Vite + TypeScript + Tailwind • Node.js + Express + WebSocket + ethers.js
